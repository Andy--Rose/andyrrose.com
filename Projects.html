IoT Light Controller

Year: In Progress
Type: Web Application
Technologies: ReactJS react-bootstrap JQuery JSX/HTML CSS3 Browserify Babel Axios
Summary
Application to control in-home LED installations. Currently, the front end UI skeleton is complete and I am working on configuring the back end. The plan is to send messages from the application to an in-home gateway (Raspberry PI) that sends signals over radio (nRF24L01) devices to instruct the nodes (Arduino) on what to do. For this, I will utilize AWS IoT for secure sending of the messages to the gateway. The back end for the application will be handled by AWS API Gateway. The security for the application will be managed by AWS Cognito, which will require AWS permissions to control the lights in my home. Altogether, this application will run on zero servers.
Features
Choose device to send instructions
Power on/off lights from anywhere on any device
Select from any color and set a device
Select from presets and set a device

Alert MGMT

Year: 2016
Type: REST API
Technologies: Go
Summary
Created a stateless application to manage production alerting using templates. Application works by generating expected alerts using templates, getting existing alerts using API, and determining action based on comparison of existing alerts vs. expected alerts. Tagging features in monitoring application are referenced instead of managing a DB, allowing the application to be stateless. Deployed in AWS using ElasticBeanstalk.
Features
Alert provisioning using templates
Alert management using templates and monitoring application state
Built to be used with any automation solutions
