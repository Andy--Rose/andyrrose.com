Ski Rental Shop P.O.S. System

Length: 3 Months (Spring 2011)
Type: End-to-End
Language: C#
Role: Independent
Database: Microsoft Access DB using inline SQL statements (OLEDB)
Summary: This program is designed to run as a P.O.S. for the rental shop at a private ski school, Webbski, located at the Summit at Snoqualmie. The program is built to store information on all of the customers, as well as the equipment. It also tracks each individual payment transaction and rental event. All of the tables are linked to maximize the availability of comparisons for reference purposes. The database is changed using dynamically created SQL commands. The program is geared toward optimal speed for the user. The way that the program is designed is also to work on an older computer, as the Ski School did not have money for a new system. This is done by pre-rendering all of the information in the database at startup so that the program does not get bogged down by heavy database transactions during the busy times of the day.
Features:
  • Quick interface for high traffic volume
  • Comprehensive search for customers and equipment
  • Selection-based search for equipment
  • Automatic DB backup
  • End-of-Day sales print and inventory check
